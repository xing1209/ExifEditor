# ExifEditor

## 功能描述

ExifEditor 是一款简单实用的 EXIF 信息查看与编辑工具，支持查看图片的基本信息、GPS 数据、拍摄时间等功能。用户可以通过软件快速加载图片并对其 EXIF 信息进行编辑。

## 使用说明

### 方式一

1. 克隆或下载此代码仓库：

    ```sh
    git clone <repository_url>
    ```

2. 安装所需依赖：

    ```sh
    pip install PyQt5 PyQtWebEngine Pillow pillow-heif piexif3
    ```

3. 运行主程序：

    ```sh
    python main_win.py
    ```
